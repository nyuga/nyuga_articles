# 从粉丝重合度看b站vup的连体人程度
假设有A和B两个vup，两人之间的连体人程度的定义为：“A的粉丝中也是B的粉丝的人的比例”和“B的粉丝中也是A的粉丝的人的比例”中的最小值。
